# AI Agent Automation Assignment

Author: Rahul Chaudhary

This project demonstrates two autonomous AI agents designed to automate real-world workflows.

## Assignment 1 – Founder Research Agent
An autonomous agent that:
- Searches the internet
- Reads webpages
- Extracts information
- Generates structured reports about founders or CEOs.

Technologies Used:
- LangChain
- OpenAI
- DuckDuckGo Search
- Web scraping

## Assignment 2 – AI Developer Assistant
An AI agent that assists developers by:
- Understanding code
- Detecting bugs
- Suggesting fixes
- Generating documentation

Technologies Used:
- LangChain
- OpenAI

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run research agent
python research_agent.py

3. Run developer assistant
python dev_agent.py
